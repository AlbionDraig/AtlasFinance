# Frontend local (Streamlit)

Este frontend consume la API de Atlas Finance y contiene login/registro, gestion de movimientos y dashboard.

## Ejecutar

1. Levanta backend y base de datos.
2. Ejecuta:

```bash
streamlit run frontend/main.py
```

## Pantallas

- Auth: login y registro
- Movimientos: alta, edicion y eliminacion de transacciones
- Dashboard: KPI, tendencias y tablas
